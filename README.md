## Installation and Setup Instructions

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

* Clone the repo

  ```sh
  git clone https://github.com/szpanel/szpanel-portfolio
  ```

### Installation

* Install NPM packages

  ```sh
  npm install
  ```

* Start server

  ```sh
  npm dev
  ```
